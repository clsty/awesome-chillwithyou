## ChillPatcher
- **Description**: A BepInEx plugin to enhance the "Chill With You" game, including music feature extensions and wallpaper engine compatibility.
- **Repository**: [BeyondtheApex/ChillPatcher](https://github.com/BeyondtheApex/ChillPatcher)
- **License**: GPLv3
- **Localization**: Chinese only.
- **System Support**: Windows (supported); Linux (unknown); MacOS (unknown)
- **Features**:
  - True FLAC support for the game.
  - Batch song import and playlist management for the game.
  - Enable the game to run properly in Wallpaper Engine environment.
  - Added in-game RIME input method.

## Chill AI Mod (AI Full Voice Dialogue)
- **Description**: Add LLM + VITS + ASR based AI full voice dialogue to the game (BepInEx plugin), allowing game characters to support real-time voice and expression/action linkage.
- **Repository**: [qzrs777/AIChat](https://github.com/qzrs777/AIChat)
- **License**: MIT
- **Localization**: Chinese only.
- **System Support**: Windows (supported); Linux (supported); MacOS (unknown)

## Chill Env Sync (Real-time Weather Sync Plugin)
- **Description**: A real-time weather sync BepInEx plugin that can automatically adjust in-game environmental effects based on real-world weather conditions, or simulate day-night cycles based on real time.
- **Repository**: [Small-tailqwq/RealTimeWeatherMod](https://github.com/Small-tailqwq/RealTimeWeatherMod)
- **License**: MIT
- **Localization**: Chinese, English.
- **System Support**: Windows (supported); Linux (unknown); MacOS (unknown)

## iGPU Savior (Performance and Experience Optimization Plugin)
- **Description**: A performance and experience optimization BepInEx plugin that provides mirroring, windowed, and portrait modes.
- **Repository**: [Small-tailqwq/iGPUSaviorMod](https://github.com/Small-tailqwq/iGPUSaviorMod)
- **License**: MIT
- **Localization**: Chinese only.
- **System Support**: Windows (supported); Linux (unknown); MacOS (unknown)

## Chill Music Information Sync (Music Information Sync)
- **Description**: A BepInEx plugin that enables bidirectional synchronization with the Windows Media Control API (SMTC).
- **Repository**: [Cainongw/ChillMusicInformationSyncMod](https://github.com/Cainongw/ChillMusicInformationSyncMod)
- **License**: MIT
- **Localization**: Chinese only.
- **System Support**: Windows (supported); Linux (unknown); MacOS (unknown)

## Chill With Anyone

- **Repository**: [Cavibot/chill-lofi-appearance-mod](https://github.com/Cavibot/chill-lofi-appearance-mod)
