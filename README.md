《放松时光：与你共享 Lo-Fi 故事》是一款主打工作/学习陪伴类型的游戏，此为与之有关的项目列表。

**声明**
- 本项目与游戏官方无关。
- 本项目与列表中的各项目官方无关。
- 本项目收录的信息尽可能做到准确，但仍有可能错漏或过时。
- 本项目不对列表中任何项目的功能与安全性作任何担保。

**若想添加其他项目，或想对已有信息进行更新，欢迎提交 PR。**

## ChillPatcher
- **简介**：这是一个 BepInEx 插件，用于增强《Chill With You》游戏. 包括音乐功能扩展和 wallpaper engine 兼容。
- **仓库**：[BeyondtheApex/ChillPatcher](https://github.com/BeyondtheApex/ChillPatcher)
- **许可**：GPLv3
- **本地化**：仅中文。
- **系统支持**：Windows（支持）；Linux（未知）；MacOS（未知）
- **特性列举**：
  - 为游戏真正的 flac 支持。
  - 为游戏提供歌曲批量导入和歌单管理。
  - 使游戏正确在 Wallpaper Engine 环境下运行。
  - 添加了游戏内 RIME 输入法。

## Chill AI Mod（AI 全语音对话）
- **简介**：为游戏添加基于 LLM + VITS + ASR 的 AI 全语音对话（BepInEx 插件），让游戏角色支持实时语音与表情动作联动。
- **仓库**：[qzrs777/AIChat](https://github.com/qzrs777/AIChat)
- **许可**：MIT
- **本地化**：仅中文。
- **系统支持**：Windows（支持）；Linux（支持）；MacOS（未知）

## Chill Env Sync（实时天气同步插件）
- **简介**：实时天气同步的 BepInEx 插件，可以根据真实世界的天气情况自动调整游戏内的环境效果，或基于现实时间模拟昼夜循环。
- **仓库**：[Small-tailqwq/RealTimeWeatherMod](https://github.com/Small-tailqwq/RealTimeWeatherMod)
- **许可**：MIT
- **本地化**：中文、英文。
- **系统支持**：Windows（支持）；Linux（未知）；MacOS（未知）
